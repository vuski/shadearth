이 프로젝트는 구형의 지구본에 실사 렌더링을 하는 프로젝트야.

1. 지구 구체를 만들고 https://github.com/NASA-AMMOS/3DTilesRendererJS 이걸 기본으로 사용할꺼야.
2. 렌더링 방식은 전적으로 아래 사이트의 기술들을 가져올꺼야. 이건 서버 코드일텐데, 클라이언트에서도 사용 가능한지 검토해줘
   https://github.com/wwwtyro/map-tile-lighting-demo/tree/master
3. mapzen dem xyz 타일과 osm xyz vector 타일을 읽어와서 실사 렌더링 하는 프로젝트야. 일단은 mapzen dem xyz 타일만 이용하자.
   주소는 다음과 같아. 맵젠 타일 형식이야.
   const TILE_SERVER = {
   BASE_URL: "https://[untitled].synology.me",
   DEM_PATH: "terrain_z10_512px/{z}/{x}/{y}.png",
   };
4. 지구본을 컨트롤하는 툴은 다음의 문서를 참고해

- .readme\20260308_3dtiles-globe-guide.md

5. 렌더링 방식은 다음을 참고해

- .readme\20260308*DEM_Raymarching*기법\_조사.md

6. 작동 방식은 다음과 같아.

- 지구본을 조작하다가 멈추면 렌더링 시작
- 타일이 모두 로드되면 한 판의 텍스쳐를 만들기(옆 타일의 dem을 참고하려면 그래야 하지 않을까?. 위의 wwwtyro 링크에 더 좋은 아이디어가 있으면 그렇게 해줘)
- 텍스쳐로 wwwtyro 방식으로 렌더링 시작
