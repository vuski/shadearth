# ShadEarth - 지구본 실사 렌더링 프로젝트 계획

## Context

구형 지구본에 실사 렌더링을 하는 프로젝트.
- 3DTilesRendererJS의 XYZTilesPlugin으로 지구본 기본 구조 구현
- wwwtyro의 Advanced Map Shading 기법(hillshading, soft shadow, AO)을 클라이언트에서 적용
- Mapzen DEM XYZ 타일을 사용한 지형 렌더링

핵심 요구사항:
1. 지구본 조작 중에는 기본 렌더링
2. 조작 멈춤 → 타일 로드 완료 → 실사 렌더링 시작
3. wwwtyro 방식의 ray marching + soft shadow + AO 적용

---

## 기술 스택

```
- Framework: Vite + TypeScript
- 3D Engine: Three.js
- Globe: 3d-tiles-renderer (XYZTilesPlugin, GlobeControls)
- Shader: GLSL (Custom ShaderMaterial)
```

## wwwtyro 스타일 렌더링

### Multi-Pass 렌더링 파이프라인

```
Pass 1: DEM → Normal Map 생성
Pass 2: Normal + Sun Direction → Hillshading
Pass 3: Ray Marching → Soft Shadow (Monte Carlo 샘플링)
Pass 4: Ray Marching → Ambient Occlusion
Pass 5: 합성 → 최종 출력
```

### 핵심 셰이더 알고리즘

**노멀 계산:**
```glsl
vec3 dx = vec3(pixelScale, 0.0, getHeight(uv + vec2(1,0)) - getHeight(uv));
vec3 dy = vec3(0.0, pixelScale, getHeight(uv + vec2(0,1)) - getHeight(uv));
vec3 normal = normalize(cross(dx, dy));
```

**Soft Shadow:** Amanatides-Woo 알고리즘으로 텍스처 공간에서 2D ray traversal

**Ambient Occlusion:** 표면 노멀 주변 반구에서 랜덤 ray 발사, 지형에 막히지 않은 비율 = AO 값

## Progressive Rendering (점진적 렌더링)

렌더링 중 매 프레임마다 화면에 결과 표시

```typescript
class ProgressiveRenderer {
  private accumRT: THREE.WebGLRenderTarget;
  private tempRT: THREE.WebGLRenderTarget;
  private frameCount = 0;

  renderFrame() {
    this.renderSample(this.tempRT, this.frameCount);
    const weight = 1.0 / (this.frameCount + 1);
    this.blend(this.accumRT, this.tempRT, weight);
    this.displayToScreen(this.accumRT);
    this.frameCount++;
  }

  reset() {
    this.frameCount = 0;
  }
}
```

**사용자 경험:**
```
[조작 멈춤]
  → Frame 1: 노이즈 심한 초기 결과 즉시 표시
  → Frame 2-10: 빠르게 개선됨
  → Frame 10-64: 점점 부드러워짐
  → Frame 64+: 고품질 최종 결과
[조작 재개]
  → 즉시 기본 렌더링으로 복귀 (누적 리셋)
```

### 태양 위치 계산 (날짜/시각 기반)

**JavaScript (전역):** 날짜/시각 → 태양 방향 벡터 (ECEF 좌표계)
```typescript
// src/utils/sunPosition.ts
interface SunDirection {
  // 태양의 3D 방향 벡터 (지구 중심 기준, Y-up)
  direction: THREE.Vector3;
  // 태양 적위 (연중 변화, -23.44° ~ +23.44°)
  declination: number;
}

function calculateSunDirection(date: Date): SunDirection {
  // 1. 율리우스 날짜 계산
  // 2. 태양 적위 (declination) 계산 - 연중 지구 기울기 반영
  // 3. 시간각 (hour angle) → 경도 오프셋
  // 4. ECEF 방향 벡터 반환
}
```

**셰이더 (픽셀별):** 표면 노멀 vs 태양 벡터 → 로컬 태양 고도
```glsl
// terrain.frag.glsl
uniform vec3 uSunDirection;  // 전역 태양 방향 벡터

void main() {
  // 구체 표면의 노멀 (= normalize(worldPosition))
  vec3 surfaceNormal = normalize(vWorldPosition);

  // 해당 위치에서의 태양 고도 = dot(노멀, 태양방향)
  float sunElevation = dot(surfaceNormal, uSunDirection);

  // sunElevation > 0: 낮 (태양이 지평선 위)
  // sunElevation < 0: 밤 (태양이 지평선 아래)
  // sunElevation = 0: 일출/일몰 라인
}
```

**이유:** 평면 지도와 달리 지구본에서는 같은 시각에도 위치마다 태양 고도가 다름
